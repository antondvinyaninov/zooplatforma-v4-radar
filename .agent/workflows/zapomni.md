# Workflow: Команды памяти (MemPalace)

// turbo-all

## 1. Сохранение диалогов (Команды "запомни", "/memory")

При вызове пользователем слова **"запомни"** (без указания пути), вы как AI-агент должны сохранить логи общения. Выполните:

1. Вызовите `run_command` с `SafeToAutoRun=true`:
   ```bash
   python3 -m mempalace mine /Users/dvinyaninov/.gemini/antigravity/brain/ --mode convos
   ```
2. Дождитесь успешного завершения сканирования.
3. Коротко отчитайтесь, что текущий диалог сохранен в "чертоги разума" MemPalace.

## 2. Добавление нового проекта (Команда "запомни проект [ПУТЬ]")

Если пользователь говорит **"запомни проект [ПУТЬ]"**, вы автоматически добавляете его исходный код в базу:

1. Выполните инициализацию проекта:
   ```bash
   python3 -m mempalace init [ПУТЬ]
   ```
   *(При необходимости используйте инструмент `send_command_input` с вводом `\n`, чтобы принять структуру папок по умолчанию).*
2. Как только инициализация завершена, запустите индексацию:
   ```bash
   python3 -m mempalace mine [ПУТЬ]
   ```
3. Сообщите пользователю об успешном сохранении файлов проекта в память.

### 💡 Краткая памятка для пользователя (Cheat Sheet)
Если пользователь просто спрашивает инструкцию "Как добавлять проекты", выведите ему эти два шага для исполнения в терминале:
1. Создать конфиг: `python3 -m mempalace init /Users/dvinyaninov/Проекты/название_проекта`
2. Отправить в базу: `python3 -m mempalace mine /Users/dvinyaninov/Проекты/название_проекта`
