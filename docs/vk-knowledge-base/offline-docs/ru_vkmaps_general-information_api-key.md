# Использование API VK Карт

[Библиотеки Maps SDK](/ru/vkmaps/general-information/general#%D0%9F%D0%BE%D0%B4%D0%B4%D0%B5%D1%80%D0%B6%D0%B8%D0%B2%D0%B0%D0%B5%D0%BC%D1%8B%D0%B5%20%D0%BF%D0%BB%D0%B0%D1%82%D1%84%D0%BE%D1%80%D0%BC%D1%8B) используются для отображения карт в приложениях. Задачи, требующие вычислений, например построение маршрутов, определение области достижимости,поиск мест и другие — проводятся с помощью API VK Карт.

## [](#Требования)Требования

Для работы с API потребуется ключ доступа. Он идентифицирует клиента, который обращается к геосервисам.

Чтобы получить ключ, отправьте заявку на странице:

    ► [https://maps.vk.com/ru/welcome/](https://vk.com/away.php?to=https%3A%2F%2Fmaps.vk.com%2Fru%2Fwelcome%2F)

При необходимости вы можете запросить несколько ключей.

## [](<#Базовый URL и формат запросов>)Базовый URL и формат запросов

Для работы с API VK Карт, используйте GET-запросы. Отправляйте их по следующему адресу:

`https://maps.vk.com/api/{endpoint}?api_key={your-key}&{params}`

*   •
    
    `{endpoint}` — вызываемая операция. Возможные значения указаны в документации по операции.
    
*   •
    
    `api_key={your-key}` — параметр, который содержит используемый ключ доступа.
    
*   •
    
    `params` — параметры запроса в виде `параметр=значение`. Параметры описаны в документации по выполняемой операции. В URL параметры разделяются символом `&`.
    

## [](<#Демонстрационный доступ>)Демонстрационный доступ

Если у вас пока нет ключа доступа, вы можете опробовать работу с API VK Карт с помощью [демонстрационного сервера](/ru/vkmaps/general-information/general#%D0%94%D0%B5%D0%BC%D0%BE%D1%81%D0%B5%D1%80%D0%B2%D0%B5%D1%80).

## [](<#Ограничение частоты запросов>)Ограничение частоты запросов

Чтобы предотвратить перегрузку серверов, мы ограничиваем скорость отправки запросов с одним и тем же ключом. По умолчанию можно отправить не более 50 запросов в секунду.

Вы можете увеличить или уменьшить этот лимит для ключа. [Обратитесь к вашему менеджеру со стороны VK Карт](/ru/vkmaps/general-information/general#%D0%A2%D0%B5%D1%85%D0%BD%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F%20%D0%BF%D0%BE%D0%B4%D0%B4%D0%B5%D1%80%D0%B6%D0%BA%D0%B0).

## [](<#Статистика использования>)Статистика использования

Мы собираем статистику использования геосервисов. Сведения формируются в течение 2 рабочих дней месяца, следующего за отчетным.

Сформированные отчёты отправляются в электронном виде лицу, которое было указано в договоре. На основании отчётов в дальнейшем формируются счета на оплату и другие документы.

## [](<#Помощь при работе с ключами доступа>)Помощь при работе с ключами доступа

Чтобы получить информацию об имеющихся у вас ключах доступа, добавить новый, удалить существующий или настроить ограничения, [обратитесь к нашим сотрудникам](/ru/vkmaps/general-information/general#%D0%A2%D0%B5%D1%85%D0%BD%D0%B8%D1%87%D0%B5%D1%81%D0%BA%D0%B0%D1%8F%20%D0%BF%D0%BE%D0%B4%D0%B4%D0%B5%D1%80%D0%B6%D0%BA%D0%B0).
