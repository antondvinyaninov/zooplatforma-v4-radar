# 5.88

*   •
    
    Метод [`messages.removeChatUser`](/ru/messages.removeChatUser) возвращает ошибку `935: User not found in chat`, если пользователя нет в чате.
    

### Связанные методы

[messages.removeChatUser](/method/messages.removeChatUser)

Исключает из мультидиалога пользователя, если текущий пользователь или сообщество является администратором беседы либо текущий пользователь пригласил исключаемого пользователя.
