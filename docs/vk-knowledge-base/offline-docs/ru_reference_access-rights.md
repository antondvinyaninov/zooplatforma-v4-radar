# Права доступа

Права доступа определяют возможность использования токена для работы с тем или иным разделом данных. Так, например, для получения номера телефона пользователя нужно получить токен с правами `phone`.

Если вы используете авторизацию VK ID OAuth, укажите строковые значения через запятую, а не битовую маску. Подробнее — в документации [VK ID](https://vk.com/away.php?to=https%3A%2F%2Fid.vk.com%2Fabout%2Fbusiness%2Fgo%2Fdocs%2Fru%2Fvkid%2Flatest%2Fvk-id%2Fconnection%2Fapi-integration%2Fapi-description%23Dostup-prilozheniya-k-dannym-polzovatelya).

На странице с описанием каждого метода указано, каких прав доступа он требует. Обратите внимание, некоторые методы не требуют определенных прав доступа, но не могут быть вызваны без `access_token`. В таком случае при получении токена вы можете не передавать параметр `scope`.

## [](<#Права доступа для токена пользователя>)Права доступа для токена пользователя

Если вы создали приложение в [Сервисе авторизации VK ID](https://vk.com/away.php?to=https%3A%2F%2Fid.vk.com%2Fabout%2Fbusiness%2Fgo%2F), изучите список прав доступа в [документации VK ID](https://vk.com/away.php?to=https%3A%2F%2Fid.vk.com%2Fabout%2Fbusiness%2Fgo%2Fdocs%2Fru%2Fvkid%2Flatest%2Fvk-id%2Fconnection%2Fwork-with-user-info%2Fscopes). Чтобы получить права из списка ниже, напишите на почту [devsupport@corp.vk.com](mailto:devsupport@corp.vk.com). Для получения `phone` [подтвердите профиль бизнеса](https://vk.com/away.php?to=https%3A%2F%2Fid.vk.com%2Fabout%2Fbusiness%2Fgo%2Fdocs%2Fru%2Fvkid%2Flatest%2Fvk-id%2Fconnection%2Fcreate-application%23Rasshirennye-dostupy).

Если вы уже запрашивали доступы из списка прав ниже для вашего приложения, делать ничего не нужно — выданные доступы сохраняются.
