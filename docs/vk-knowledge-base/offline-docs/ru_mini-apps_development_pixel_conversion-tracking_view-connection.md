# Проверка, что событие установлено корректно

Для диагностики:

1.  1.
    
    Откройте страницу сайта, на которой установлен пиксель.
    
2.  2.
    
    Откройте **Console** в браузере. Для этого просмотрите код страницы; можно воспользоваться комбинацией клавиш:
    
    *   •
        
        `Ctrl + Shift + I` для Windows.
        
    *   •
        
        `⌥ + ⌘ + I` для macOS.
        
    
3.  3.
    
    Перейдите на вкладку **Cеть** (**Network**).
    
4.  4.
    
    Отфильтруйте вызовы по слову `rtrg`. В результате вы будете наблюдать все вызовы пикселя в конверсии.
    
    *   •
        
        Название события, в данном случае «Конверсия».

    *   •
        
        Ценность события, в данном случае `100`.
