# Разработка мини-приложений ВКонтакте

![](https://sun9-35.userapi.com/5BofrXJaPtqlPPJYpvLGEqZM6qbdVq5CItBS9g/4rdsc_nkWks.jpg)

Этот видеокурс рассказывает, как создавать мини-приложения ВКонтакте. В нём мы рассматриваем все важные этапы: от поиска идеи и создания дизайна до разработки, монетизации и продвижения. На каждом шаге эксперты платформы VK Mini Apps делятся своим опытом и дают практические советы.

Для демонстрации идей мы используем мини-приложение «Блюдо дня». Оно показывает карту со списком кафе и ресторанов, в которых пользователь может заказать понравившееся блюдо.
