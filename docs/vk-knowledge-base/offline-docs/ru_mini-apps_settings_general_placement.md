# Размещение (настройки мини-приложений)

Раздел содержит настройки, которые необходимы для запуска мини-приложения из интерфейса площадок ВКонтакте и Одноклассники.

**Важно!** Если настройки размещения не указаны, мини-приложение не будет работать.

**Чтобы открыть раздел:**

1.  1.
    
    Перейдите в [список приложений](https://vk.com/away.php?to=https%3A%2F%2Fdev.vk.com%2Fru%2Fadmin%2Fapps-list).
    
2.  2.
    
    Откройте настройки нужного мини-приложения.
    
3.  3.
    
    В меню слева выберите **Размещение**.
    

## [](#Доступ)Доступ

Редактировать настройки размещения может:

*   •
    
    Создатель мини-приложения.
    
*   •
    
    Администратор с полным доступом.
    
*   •
    
    Администратор с частичным доступом, с правом «Редактирование».
    

Подробнее о правах — в разделе [Администраторы](/ru/mini-apps/settings/managers).

## [](<#Состояние для пользователей>)Состояние для пользователей

Настройка указывает, кому доступно мини-приложение

Состояние

Описание

**Отключено**

Состояние мини-приложения сразу после создания. Приложение с таким состоянием доступно только администраторам. Они могут запускать мини-приложение по прямой ссылке `https://vk.com/app{id}`. Чтобы опубликовать приложение в Одноклассниках, нужно привязать OK ID в разделе [Другие площадки](/ru/mini-apps/settings/general/other-sites). Мини-приложение станет доступно в Одноклассниках в течение 15 минут.

**Включено**

Мини-приложение ВКонтакте станет доступно по ссылке всем пользователям. В Одноклассниках доступ будет только у создателя приложения. Чтобы приложение стало доступно всем пользователям, нужно пройти [модерацию](/ru/mini-apps/settings/moderation).

Подробнее о размещении приложения в Одноклассниках — в разделе [Публикация в Одноклассниках](/ru/ok/publishing).

## [](<#Настройки платформ>)Настройки платформ

Раздел содержит настройки запуска мини-приложения для различных платформ ВКонтакте. Вы можете настроить запуск приложения на всех доступных платформах или выбрать только необходимые.

Раздел

Описание

**Мобильное приложение**

Настройки для запуска мини-приложения из мобильного приложения ВКонтакте и Одноклассники для Android и iOS. Приложение будет открываться в элементе управления WebView и будет выглядеть похожим на обычные приложения для Android и iOS.

**Десктопная версия сайта**

Настройки для запуска мини-приложения из десктопной версии сайта ВКонтакте ([vk.com](https://vk.com/away.php?to=https%3A%2F%2Fvk.com)) и Одноклассники ([ok.ru](https://vk.com/away.php?to=https%3A%2F%2Fok.ru)). В браузере у пользователя будет открыта страница сайта vk.com, а приложение будет загружено в HTML-элемент `<iframe>` на этой странице.

**Мобильная версия сайта**

Настройки для запуска мини-приложения из мобильной версии сайта ВКонтакте ([m.vk.com](https://vk.com/away.php?to=https%3A%2F%2Fm.vk.com)) и Одноклассники ([m.ok.ru](https://vk.com/away.php?to=https%3A%2F%2Fm.ok.ru)). Приложение будет открываться в элементе управления WebView и будет выглядеть похожим на обычные приложения для Android и iOS.

### [](#URL)URL

Адрес для запуска мини-приложения. Платформа отправит этот адрес в WebView или в `<iframe>` для загрузки.

Чтобы передать параметры, укажите их в URL после символа `?`, например `https://my-app.com/start?param1=value1`.

### [](<#Режим разработки>)Режим разработки

Используйте эту настройку, чтобы включить или отключить режим разработки. В поле укажите адрес, по которому будет доступна версия игры только для администраторов.

![Настройка режима разработки](https://sun9-55.userapi.com/n2hDr2y9p_aqZdwxMTFhoQVLB5ayCpDbHEfmbQ/jHK5TuJvZDE.jpg "Настройка режима разработки")Настройка режима разработки

Режим разработки полезен, когда мини-приложение уже опубликовано в каталоге, но вам надо проверить изменения в нём. Этот режим работает для всех [администраторов](/ru/mini-apps/settings/managers) мини-приложения. Когда они запустят мини-приложение, платформа ВКонтакте загрузит его из адреса, указанного в настройке **Режим разработки**, а не из настройки **URL**. Для остальных пользователей мини-приложение продолжит открываться по **URL**, для них подмены не произойдёт.

### [](<#Приоритеты URL для загрузки мини-приложения>)Приоритеты URL для загрузки мини-приложения

Когда администратор запускает мини-приложение, URL для загрузки определяется в следующем порядке.

1.  1.
    
    URL, указанный в настройках [тестовой группы](/ru/mini-apps/settings/test-groups), если администратор включён в какую-либо тестовую группу. Этот URL имеет наивысший приоритет относительно остальных.
    
2.  2.
    
    URL, указанный для режима разработки, если режим разработки включён для выбранной платформы.
    
3.  3.
    
    URL, указанный в настройке [URL](#URL). Приоритет этого URL ниже адресов, указанных в тестовой группе или в режиме разработки.
    

## [](<#Удалить приложение>)Удалить приложение

Чтобы удалить мини-приложение с платформы VK Mini Apps, в правом нижнем углу страницы нажмите **Удалить приложение**. Кнопка доступна только [создателю](/ru/mini-apps/settings/managers#%D0%A1%D0%BE%D0%B7%D0%B4%D0%B0%D1%82%D0%B5%D0%BB%D1%8C) мини-приложения.

> Удалённое мини-приложение нельзя восстановить. Можно создать новое с другим идентификатором.
