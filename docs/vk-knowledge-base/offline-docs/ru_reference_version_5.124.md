# 5.124

*   •
    
    В методе [`groups.get`](/ru/groups.get) в объекте `market` приходит поле `type (string)`, которое может иметь значения `basic` и `advanced` в зависимости от типа магазина. Поле приходит, если в группе включен раздел «Товары».
    

### Связанные методы

[groups.get](/method/groups.get)

Возвращает список сообществ указанного пользователя.
