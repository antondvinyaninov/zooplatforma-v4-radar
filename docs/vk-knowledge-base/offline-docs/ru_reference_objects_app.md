# Приложение

Объект, описывающий приложение, содержит следующие поля:

## [](<#Базовые поля>)Базовые поля

### [](#id)`id`

`integer` Идентификатор приложения.

### [](#title)`title`

`string` Название приложения.

### [](#icon_278)`icon_278`

`string` URL-адрес обложки приложения шириной 278px.

### [](#icon_139)`icon_139`

`string` URL-адрес обложки приложения шириной 139px.

### [](#icon_150)`icon_150`

`string` URL-адрес обложки приложения шириной 150px.

### [](#icon_75)`icon_75`

`string` URL-адрес обложки приложения шириной 75px.

### [](#banner_560)`banner_560`

`string` URL-адрес баннера шириной 560px.

### [](#banner_1120)`banner_1120`

`string` URL-адрес баннера шириной 1120px.

### [](#type)`type`

`string` Тип приложения. Возможные значения:

*   •
    
    `vk_app` — мини-приложение ВКонтакте / VK;
    
*   •
    
    `app` — социальное приложение;
    
*   •
    
    `game` — игра;
    
*   •
    
    `site` — подключаемый сайт;
    
*   •
    
    `standalone` — отдельное приложение (для мобильного устройства).
    

### [](#section)`section`

`string` Категория приложения.

### [](#author_url)`author_url`

`string` Адрес страницы автора приложения.

### [](#author_id)`author_id`

`integer` Идентификатор автора приложения.

### [](#author_group)`author_group`

`integer` Идентификатор официальной группы приложения.

### [](#members_count)`members_count`

`integer` Количество участников приложения.

### [](#published_date)`published_date`

`integer` Дата размещения в Unixtime.

### [](#catalog_position)`catalog_position`

`integer` Позиция в каталоге.

### [](#international)`international`

`integer` Является ли приложение мультиязычным (_1_).

### [](#leaderboard_type)`leaderboard_type`

`integer` Тип таблицы результатов. Возможные значения:

*   •
    
    `0` — не поддерживается;
    
*   •
    
    `1` — по уровню;
    
*   •
    
    `2` — по очкам.
    

### [](#genre_id)`genre_id`

`integer` Идентификатор жанра

### [](#genre)`genre`

`string` Название жанра

### [](#platform_id)`platform_id`

`string` Идентификатор приложения в магазине приложений

### [](#is_in_catalog)`is_in_catalog`

`integer` Доступно ли приложение в мобильном каталоге. Возможные значения:

*   •
    
    `1` — доступно;
    
*   •
    
    `0` — недоступно.
    

### [](#friends)`friends`

`array` Список идентификаторов друзей текущего пользователя, которые установили приложение (если был передан параметр `return_friends` = `1`.

### [](#installed)`installed`

`integer | boolean` `1`, если приложение установлено у текущего пользователя.

### [](#is_html5_app)`is_html5_app`

`integer` `1`, если приложение — HTML5-игра.

### [](#screen_orientation)`screen_orientation`

`integer` Поддерживаемая ориентация экрана. Возможные значения:

*   •
    
    `0` — альбомная и портретная;
    
*   •
    
    `1` — только альбомная;
    
*   •
    
    `2` — только портретная.
    

### [](#mobile_controls_type)`mobile_controls_type`

`integer` Тип элементов управления игрой на мобильных устройствах. Возможные значения:

*   •
    
    `1` — прозрачный элемент управления поверх области с игрой;
    
*   •
    
    `0` — чёрная полоска над областью с игрой;
    

### [](#mobile_view_support_type)`mobile_view_support_type`

`integer` `1`, если игра использует нижнюю часть экрана на iPhoneX

## [](<#Опциональные поля>)Опциональные поля

Объект содержит опциональные поля, если в методе был передан параметр `extended` = `1`.

### [](#description)`description`

`string` Описание.

### [](#screen_name)`screen_name`

`string` Короткий адрес приложения (или строка `idXXXXXXX`, если короткий адрес не задан).

### [](#icon_16)`icon_16`

`string` URL-адрес обложки приложения шириной 16px.

### [](#screenshots)`screenshots`

`array` Массив объектов [фотографий](/ru/reference/objects/photo), описывающих скриншоты приложения.

### [](#push_enabled)`push_enabled`

`integer` `1`, если у пользователя включены уведомления из этого приложения.
