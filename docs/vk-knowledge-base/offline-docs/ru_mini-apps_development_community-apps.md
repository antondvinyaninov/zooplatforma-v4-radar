# Плагины для сообществ

Плагины — это мини-приложения, которые работают только в контексте сообщества и расширяют его функции. Плагины можно монетизировать, продвигать с помощью рекламы и размещать в [каталоге плагинов](https://vk.com/away.php?to=https%3A%2F%2Fvk.com%2Fcommunity_apps).

С помощью плагинов вы можете реализовать новые возможности в сообществе, например добавить форму для записи на услугу, создать лендинг для подписчиков сообщества или создать список для регулярных рассылок сообщений подписчикам.

В [каталог плагинов](https://vk.com/away.php?to=https%3A%2F%2Fvk.com%2Fcommunity_apps) попадают приложения, которые прошли модерацию и могут быть установлены в сообществе. Подробно о модерации и преимуществах прохождения проверки — в разделе [Модерация плагинов](/ru/mini-apps/development/community-apps/creating-and-settings/#%D0%9C%D0%BE%D0%B4%D0%B5%D1%80%D0%B0%D1%86%D0%B8%D1%8F).

**Важно!** Размещение плагинов для сообществ в Одноклассниках через платформу VK Mini Apps недоступно. Приложения для групп в Одноклассниках создаются непосредственно на сайте ok.ru. Подробнее — в документации [API Одноклассники](https://vk.com/away.php?to=https%3A%2F%2Fapiok.ru%2Fapps%2Fgroups).

## [](#Особенности)Особенности

*   •
    
    Плагин для сообщества — это отдельный тип мини-приложений.
    
*   •
    
    Плагин работает только в контексте сообщества.
    
*   •
    
    Если у пользователя нет доступа к закрытому или частному сообществу, плагин по прямой ссылке откроется без контекста сообщества.
    
*   •
    
    Установленный плагин можно запустить из сообщества, поста в сообществе или из рекламного объявления.
    
*   •
    
    Установить плагин можно по ссылке или из каталога.
    
*   •
    
    Плагин может быть размещён только в каталоге плагинов.
    
*   •
    
    В сообщество можно добавить не более 5 плагинов.
    

## [](#Рекомендации)Рекомендации

Мы собрали рекомендации, которые помогут улучшить ваш плагин для сообщества.

#### [](<#Главная страница>)Главная страница

*   •
    
    При запуске вне контекста сообщества, когда отсутствует параметр запуска [`vk_group_id`](/ru/mini-apps/development/launch-params), правильно показывайте главную страницу с описанием плагина и кнопкой для его установки. Используйте событие [`VKWebAppAddToCommunity`](/ru/bridge/VKWebAppAddToCommunity), которое предлагает установить плагин в сообщество.
    
    Примеры приложений с главной страницей:
    
    *   •
        
        [Spycat — виджеты для продаж](https://vk.com/away.php?to=https%3A%2F%2Fvk.com%2Fapp5686299)
        
    *   •
        
        [Виджет в сообществе](https://vk.com/away.php?to=https%3A%2F%2Fvk.com%2Fapp6244292)
        
    
*   •
    
    Если мини-приложение запускается в контексте сообщества, по ссылке вида `vk.com/app{ИДЕНТИФИКАТОР_МИНИ_ПРИЛОЖЕНИЯ}_-{ИДЕНТИФИКАТОР_СООБЩЕСТВА}`, отлавливайте параметр запуска [`vk_group_id`](/ru/mini-apps/development/launch-params) и отображайте релевантный контент для текущего пользователя в зависимости от его роли в сообществе.
    

#### [](<#Разграничение доступа>)Разграничение доступа

Используйте параметр запуска [`vk_viewer_group_role`](/ru/mini-apps/development/launch-params), чтобы разграничить доступ к приложению в зависимости от роли пользователя в сообществе.

Зная роль пользователя, вы можете:

*   •
    
    Показывать настройки приложения только администраторам сообщества.
    
*   •
    
    Открыть некоторые функции только для редакторов или модераторов сообщества.
    
*   •
    
    Показывать определённый контент только для подписчиков сообщества.
    

> **Обратите внимание!** Нельзя ограничивать доступ к части функций до того, как пользователь подпишется на сообщество.

#### [](#Сниппеты)Сниппеты

Используйте [сниппеты](/ru/mini-apps/settings/general/design#%D0%98%D0%BA%D0%BE%D0%BD%D0%BA%D0%B0%20%D0%B4%D0%BB%D1%8F%20%D0%BA%D0%B0%D1%82%D0%B0%D0%BB%D0%BE%D0%B3%D0%B0%20%D0%B8%20%D1%81%D0%BD%D0%B8%D0%BF%D0%BF%D0%B5%D1%82%D0%BE%D0%B2), чтобы сделать ссылку на плагин персонализированной.

#### [](<#Сообщения сообщества>)Сообщения сообщества

Используйте сообщения сообщества, чтобы уведомлять пользователя о чём-либо после того, как он вышел из приложения.

Отправлять сообщения можно как пользователям, так и администраторам сообществ. Например: плагин может отправлять сообщение каждому, кто заполнил форму, и уведомлять администратора сообщества о новой заявке.

К сообщению можно прикреплять изображения, документы, видео, аудио и точку на карте.

Подробнее — в разделе [Сообщения сообществ](/ru/widgets/community-messages).

#### [](#Виджеты)Виджеты

Добавьте возможность устанавливать виджет плагина. Виджет — это блок данных, отрисованный в соответствии с формальным описанием, которое предоставляет плагин для сообщества.

Разместите в нём часть информации из приложения, пользователь увидит её прямо на странице сообщества. Выберите подходящий шаблон в зависимости от ваших потребностей.

Содержимое виджета можно обновлять. Если информация в плагине часто меняется, вы можете поддерживать актуальную информацию и в виджете.

*   •
