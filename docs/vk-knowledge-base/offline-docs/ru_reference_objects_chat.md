# Чат

Объект, описывающий чат, содержит следующие поля:

## [](#id)`id`

`integer` Идентификатор беседы.

## [](#type)`type`

`string` Тип диалога.

## [](#title)`title`

`string` Название беседы.

## [](#admin_id)`admin_id`

`integer` Идентификатор пользователя, который является создателем беседы.

## [](#users)`users`

`array` Список идентификаторов (`integer`) участников беседы.

## [](#push_settings)`push_settings`

`object` Настройки оповещений для диалога. Объект, содержащий поля:

*   •
    
    `sound` (`integer`) — указывает, включен ли звук оповещений (`1` — включен, `0` — отключен);
    
*   •
    
    `disabled_until` (`integer`) — указывает, до какого времени оповещения для чата отключены. `-1` — отключены навсегда (бессрочно).
    

## [](#photo_50)`photo_50`

`string` URL изображения-обложки чата шириной 50 px (если доступно).

## [](#photo_100)`photo_100`

`string` URL изображения-обложки чата шириной 100 px (если доступно).

## [](#photo_200)`photo_200`

`string` URL изображения-обложки чата шириной 200 px (если доступно).

## [](#photo_base)`photo_base`

`string` URL изображения-обложки чата шириной 400 px (если доступно). Добавив GET-параметр `cs=<size>x<size>`, где size=`50`|`100`|`200` можно получить изображение соответствующего размера.

## [](#left)`left`

`integer` Флаг, указывающий, что пользователь покинул беседу. Всегда содержит `1`.

## [](#kicked)`kicked`

`integer` Флаг, указывающий, что пользователь был исключен из беседы. Всегда содержит `1`.
