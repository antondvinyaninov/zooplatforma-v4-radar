# 5.27

В метод `notifications.get` добавлен новый параметры `start_from`. В возвращаемый методом объект добавлено поле `next_from`.

### Связанные методы

[notifications.get](/method/notifications.get)

Возвращает список оповещений об ответах других пользователей на записи текущего пользователя.
