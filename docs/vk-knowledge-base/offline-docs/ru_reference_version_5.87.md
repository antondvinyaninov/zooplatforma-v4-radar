# 5.87

*   •
    
    В объекте [сообщества](/ru/reference/objects/group) у поля `wall` поменялись значения. Для сообществ возвращаются значения:
    
    *   •
        
        `2` – стена ограничена (было `1`).
        
    *   •
        
        `3` – стена закрытая (было `0`).
