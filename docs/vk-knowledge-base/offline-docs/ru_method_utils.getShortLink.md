# utils.getShortLink

Позволяет получить URL, сокращённый с помощью vk.cc.

Для вызова метода можно использовать:

• [ключ доступа пользователя](/ru/api/access-token/getting-started#Ключ%20доступа%20пользователя)

• [ключ доступа сообщества](/ru/api/access-token/getting-started#Ключ%20доступа%20сообщества)

• [сервисный ключ доступа](/ru/api/access-token/getting-started#Сервисный%20ключ%20доступа)

## [](#Параметры)Параметры

url

string

URL, для которого необходимо получить сокращённый вариант.

Обязательный параметр

private

checkbox

`1` — статистика ссылки приватная. `0` — статистика ссылки общедоступная.

## [](#Результат)Результат

Возвращает объект, содержащий следующие поля:

### [](#short_url)`short_url`

`string` Сокращённая ссылка.

### [](#access_key)`access_key`

`string` Ключ для доступа к приватной статистике ссылки.

### [](#key)`key`

`string` Содержательная часть ссылки (после «vk.cc»).

### [](#url)`url`

`string` Оригинальный URL.
