# Образовательные курсы в Lerna

В рамках поддержки разработчиков мини-приложений команда ВКонтакте предлагает вам воспользоваться скидкой на образовательные курсы на платформе [Lerna](https://vk.com/away.php?to=https%3A%2F%2Fwww.lerna.ru).

## [](<#Почему стоит попробовать>)Почему стоит попробовать

Программа предлагает 100+ курсов по программированию, дизайну, управлению проектами, развитию коммуникативных навыков, финансовой аналитике и другим областям знаний, которые могут быть полезны создателям мини-приложений.

Список курсов доступен в личном кабинете на сайте платформы Lerna после регистрации.

Разработчики мини-приложений могут приобрести курсы из этого списка со скидками 50% и более. Предложением могут воспользоваться все члены команды, работающие над мини-приложением.

## [](#Условия)Условия

Ваше мини-приложение должно быть [опубликовано](/ru/mini-apps/catalog/getting-started) в [каталоге мини-приложений](https://vk.com/away.php?to=https%3A%2F%2Fvk.com%2Fservices).

## [](#Регистрация)Регистрация

Для участия необходимо зарегистрироваться:

    ► [Отправить заявку сейчас](https://vk.cc/ctcPWV)

После регистрации список курсов с применёнными скидками будет доступен в личном кабинете Lerna. Вводить промо-коды не нужно.
