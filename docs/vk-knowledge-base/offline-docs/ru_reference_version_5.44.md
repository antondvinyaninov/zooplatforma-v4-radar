# 5.44

*   •
    
    Методы `docs.get`, `docs.getById` возвращают поле `preview` вместо полей `photo_100` и `photo_130`.
    
*   •
    
    Метод `messages.searchDialogs` возвращает сообщества в результате поиска.
    

### Связанные методы

[docs.get](/method/docs.get)

Возвращает расширенную информацию о документах пользователя или сообщества.

[docs.getById](/method/docs.getById)

Возвращает информацию о документах по их идентификаторам.
