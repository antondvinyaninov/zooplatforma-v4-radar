[getScopes](/ru/method/apps.getScopes)

Метод получает права доступа.

Есть два вида доступов:

*   •
    
    **Базовые** — фамилия и имя, фото профиля, пол и дата рождения — доступны сразу после создания приложения в [Сервисе авторизации VK ID](https://vk.com/away.php?to=https%3A%2F%2Fid.vk.com%2Fabout%2Fbusiness%2Fgo%2F).
    
*   •
    
    **Расширенные** — доступны после [подтверждения профиля бизнеса в сервисе VK Бизнес ID](https://vk.com/away.php?to=https%3A%2F%2Fid.vk.com%2Fabout%2Fbusiness%2Fgo%2Fdocs%2Fru%2Fvkid%2Flatest%2Fvk-id%2Fconnection%2Fcreate-application%23Kak-podtverdit-profil-biznesa). Если профиль бизнеса уже подтверждён, появится доступ к номеру телефона пользователя. Чтобы метод возвращал другие [расширеннные права доступа](/ru/reference/access-rights#%D0%9F%D1%80%D0%B0%D0%B2%D0%B0%20%D0%B4%D0%BE%D1%81%D1%82%D1%83%D0%BF%D0%B0%20%D0%B4%D0%BB%D1%8F%20%D1%82%D0%BE%D0%BA%D0%B5%D0%BD%D0%B0%20%D0%BF%D0%BE%D0%BB%D1%8C%D0%B7%D0%BE%D0%B2%D0%B0%D1%82%D0%B5%D0%BB%D1%8F), запросите их через обращение в техническую поддержку. Для этого напишите на почту [devsupport@corp.vk.com](mailto:devsupport@corp.vk.com). Получение расширенных доступов рассматривается в индивидуальном порядке.
